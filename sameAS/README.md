# SameAS 时间管理应用

一个受点滴时间、时间日志、aTimeLogger、Toggl Track 启发的单页时间追踪应用原型。

## 功能

- 今日专注时钟与本周累计概览
- 项目分类卡片、颜色标签、目标时长
- 一键开始 / 暂停当前任务计时
- 时间轴日志与手动补录面板
- 简单统计：今日时长、专注率、连续记录天数、项目分布
- 本地存储：刷新页面后保留数据

## 启动

1. 安装依赖
   - `npm install`
2. 启动开发环境
   - `npm run dev`
3. 打开浏览器访问终端里显示的本地地址

## 构成

- `index.html`：页面入口
- `src/main.js`：应用逻辑与渲染
- `src/styles.css`：界面样式

## 说明

这是一个前端高保真原型，适合继续扩展为：

- 多端同步
- 账号体系
- 数据导出 / 报表
- 番茄钟提醒
- 月度 / 年度分析
